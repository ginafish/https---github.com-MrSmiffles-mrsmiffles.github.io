What is the BLS Public API?

The BLS Public API is an API released by the Bureau of Labor Statistics that gives data on the United States' economy. It offers quite a lot of data, including:

  • Employment & Unemployment
  • American Spending & Time Use
  • Inflation & Prices
  • Pay & Benefits
  • Workplace Injuries
  • American Productivity
  • International Economic Information
  • Projections for Employment

Because of the breadth of options offered by the API, when discussing results from calls, we will only focus on the results returned from one particular call at once. The only differences between calling different types of data with the BLS API are a different series_id and different return data.

You can find the official BLS API documentation here: http://www.bls.gov/developers/home.htm. Also, take a look at the Terms of Service if you plan to use the data returned from the calls in an application.

Why use this guide?

The documentation for the BLS' API mainly focuses on giving examples of code on how to use their API in different languages, on top of not covering how to make Node.js calls. This guide includes code examples for making HTML GET/POST requests in javascript using Node.js, as well as more information on structuring your calls.

How to use this guide

This guide assumes that you have a basic knowledge of Node.js, which is basically just how to install packages and run .js scripts through Node.js. If you want to learn about Node.JS, visit this guide or the official site. Examples will list what packages that they are using, but extra files not directly related to formatting the HTML requests, such as the .handlebars files, will not be displayed.
